# ssl unpinn android apps (No Frida)

1. Physical device needed
2. Root with magisk
3. Setup && Install xposed manager


After setting up all the above. Install trustmealready module in xposed
and also install proxydroid in the mobile which used to setup proxy between mobile and burp easy way.
Install the burp cert in the mobile.


`Warning` :- Installing xposed makes your mobile unable to use banking apps and also your google service may not work properly.
